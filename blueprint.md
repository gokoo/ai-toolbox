# AI工具箱项目蓝图

## 项目愿景

打造一个集成多种AI功能的工具箱，基于Lobe Chat框架进行二次开发，提供i18N翻译、AI原型图生成、AI文案改写等功能，支持多种AI模型接入，具有良好的移动端适配性，可独立部署在云服务器上。

## 技术架构

### 前端架构

- **基础框架**：基于Lobe Chat (Next.js)
- **UI框架**：Ant Design + Tailwind CSS
- **状态管理**：Zustand
- **国际化**：i18next
- **插件系统**：自定义插件架构
- **API通信**：Axios + SWR
- **部署方式**：Vercel/自托管

### 后端架构

- **基础框架**：Express.js/Nest.js
- **数据库**：MongoDB
- **认证**：JWT
- **文件存储**：本地/OSS
- **部署方式**：Docker + 阿里云/腾讯云
- **API文档**：Swagger
